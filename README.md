# Smart Voice Intelligence Console
Интерактивное ML-приложение для распознавания речи в реальном времени с анализом и метриками речи.Поддерживает исключительно русский язык.Использует PyQt6 в качестве UI.
Модель весит 50мб.

## Функционал
- Потоковое распознавание речи на русском(Vosk)
- Сегменты слов с таймкодами
- Метрики речи: скорость (WPM), паузы, filler-слова
- Экспорт JSON с результатами
